# Output Style: Ultra Concise

Maximum token efficiency. Fragments over sentences.

## Rules

1. No articles (a, an, the)
2. No filler words
3. Fragments OK
4. Lists > prose
5. Symbols > words (✓, ✗, →)

## Examples

### Status update
```
✓ tests passing
✓ lint clean
→ ready for review
```

### Error
```
✗ build failed: missing dep
```

### Answer
```
Line 42, `processUser` fn
```
